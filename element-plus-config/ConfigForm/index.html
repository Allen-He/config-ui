ConfigForm ​

配置化的表单组件。支持表单项的显示、校验、联动等常见功能，可自定义组件和布局，提供完整的类型提示。

自定义表单控件 ​

除了上述 Element Plus 内置的表单控件，component 属性支持传入自定义表单控件组件：

ts
{
  field: 'xxx',
  component: YourComponent,
  componentProps: {
    size: 'large',
    placeholder: '自定义占位符'
  },
  componentSlots: {
    append: () => <div>...</div>
  }
}
